## Where to get help

- **General questions & ideas:** [GitHub Discussions](https://github.com/LioExp/LiteML/discussions)
- **Bug reports:** Open an issue using the bug template
- **Feature requests:** Open an issue using the feature request template

## Response time

Maintainers aim to respond to issues and discussions within **7 business days**.
Security vulnerabilities have top priority — see our [Security Policy](SECURITY.md) for details.

## Before you ask

- Make sure you're using the latest version of `liteml`.
- Check the [documentation](docs/) and [Discussions](https://github.com/LioExp/LiteML/discussions) — your question may already be answered.
- If reporting a bug, include your **RAM, CPU, OS, and whether Obsidian was open**. Logs with `LITEML_DEBUG=1` help a lot.
